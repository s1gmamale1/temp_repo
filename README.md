# Agent Network Web HQ

Web HQ Dashboard for the Agent Network Platform.

## Structure
- `web-hq/` - React + TypeScript + Vite frontend
- `api-server/` - Fastify + SQLite + WebSocket backend

## Quick Start
```bash
# Backend
cd api-server && npm install && npm run dev

# Frontend  
cd web-hq && npm install && npm run dev
```
